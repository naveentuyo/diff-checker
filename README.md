# diff-checker
A standalone, single-file tool to compare two blocks of text.

## Try it Now
**[Click here to view the Live Demo](https://naveentuyo.net/diff-checker/)**

## How to Run Locally
If you prefer to run it offline:
1. Download `index.html`.
2. Open it in any web browser.

## Features
* **Real-time:** Updates instantly as you type.
* **Visuals:** Green = Added, Red = Removed.
* **Modes:** Switch between "Words" (writing) and "Characters" (code/typos).

## Tech
* **Single File:** Pure HTML/CSS/JS.
* **Library:** Uses `jsdiff` via CDN.
* **Privacy:** Runs entirely in the browser (client-side). No data is sent to a server.
